# Support

- **Bug?** Open a [bug report](https://github.com/0xzerolight/topic_watch/issues/new?template=bug_report.yml).
  Include the output of `python -m app.cli doctor` (Docker:
  `docker compose exec topic-watch python -m app.cli doctor`) — it gathers
  version, runtime, and a secret-safe config snapshot for you.
- **Idea or feature request?** Open a
  [feature request](https://github.com/0xzerolight/topic_watch/issues/new?template=feature_request.yml).
- **Question or general discussion?** Use
  [Discussions](https://github.com/0xzerolight/topic_watch/discussions).
- **Security vulnerability?** Do not open a public issue — see [SECURITY.md](SECURITY.md).

New to the project? Start with the [README](README.md) (including its
Troubleshooting table), [CONTRIBUTING.md](CONTRIBUTING.md), and
[ARCHITECTURE.md](ARCHITECTURE.md).
